## Flask Backend API

Simple REST API built using Flask to serve basic JSON responses. Designed as a lightweight backend service that can be integrated with frontend applications (e.g., portfolio, dashboards, or other web apps).

---

## 🚀 Features
- Minimal Flask setup
- JSON-based API responses
- Two endpoints:
  - Root endpoint (`/`)
  - API endpoint (`/api/hello`)
- Runs on configurable host and port

---

## 📌 Tech Stack
- Python 3
- Flask
